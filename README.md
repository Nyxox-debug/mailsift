This is a Spam/Ham email classifier that uses a model trained on the [SpamAssassin](https://www.kaggle.com/datasets/ganiyuolalekan/spam-assassin-email-classification-dataset) dataset. The model is converted to ONNX format, loaded statically, and used directly in the Svelte frontend.
